# Sales KPI Dashboard (Excel)

## Overview
This project is a dynamic Excel-based KPI dashboard designed to track and visualize core business performance metrics using linked formulas, structured data tables, and interactive KPI cards.

## Key Features
- Total Revenue tracking (automated SUM aggregation)
- Total Profit calculation
- Best Performing Region identification (dynamic ranking)
- Top Product analysis based on revenue contribution
- Interactive KPI cards using linked Excel shapes
- Fully dynamic updates based on raw data input

## Tools Used
- Microsoft Excel
- Structured Tables (Excel Tables)
- Formulas: SUM, SUMIF, UNIQUE, INDEX, MATCH
- Dashboard design principles (KPI cards, visual hierarchy)

## Dashboard Structure
- Raw_Data: stores transactional sales data
- KPI_Calculations: computes all metrics
- Dashboard: visual KPI interface

## How It Works
1. Data is entered into Raw_Data table
2. KPI_Calculations sheet processes metrics using formulas
3. Dashboard pulls live values into KPI cards via linked cells

## Skills Demonstrated
- Data analysis in Excel
- Dashboard design
- Business intelligence logic
- Formula-driven automation
- Data structuring and cleaning
